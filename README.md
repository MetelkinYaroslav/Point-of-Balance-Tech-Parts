# Point of Balance

Сайт по продаже масел и химии для хард-эндуро мотоциклов. Проект для поступления в проектную школу.

---

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid, медиазапросы)
- Google Fonts (Montserrat, Anonymous Pro)

---

## 📁 Структура проекта

point-of-balance/
├── index.html              # Главная страница
├── pages/
│   └── catalog.html        # Страница каталога
├── css/
│   ├── style.css
│   ├── catalog.css
│   └── product.css
├── assets/
│   ├── images/             # Изображения
│   └── fonts/              # Шрифты
└── README.md

---

## 📄 Страницы

- **Главная** — презентация бренда и товаров
- **Каталог** — карточки товаров с ценами и кнопкой заказа
- Остальные страницы в разработке
